##  How to Run (for Exercises with Node/React)

### For Node.js Backends (Ex 5, 6, 7):
```bash
cd Exercise-5
npm install
node server.js
```

### For MERN (Exercise 7):
**Backend:**
```bash
cd Exercise-7/server
npm install
npm run dev
```

**Frontend:**
```bash
cd Exercise-7/client
npm install
npm start
```

>  Make sure MongoDB is running locally on `mongodb://localhost:27017`
